進化のプロセスを通じて自然界に育まれた生体には、現在の工学技術では実現できない **極めて巧みで高度な生体機能** が備わっています。  
そして、そのメカニズムを解析することは、**生体機能の解明のみならず、さまざまな新しい工学システムの開発** につながる可能性があります。

私たち生体システム論研究室では、**サイエンティストの目で生体の秘密に迫る** と同時に、**エンジニアの目で人間の役に立つ機械を開発** するという二つの視点で研究に取り組んでいます。  

- 生体機能の特徴を **理論・実験の両面から工学的に解析**  
- 生体システム特有の **新たな原理を発見**  
- その原理に基づいた **医療福祉機器・産業機器の創出**  

また、こうした研究活動を通じて、**電気・電子・システム・情報工学を基礎とした生体システム論に関する深い知識** を身につけ、  
**新しい原理の追求や新分野への展開が可能な創造性を持った人材の育成** を目指しています。
